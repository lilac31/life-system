# 数据同步设置指南

## 🌐 功能介绍

你的生活管理系统现在支持数据同步功能，可以在不同设备间保持数据一致。

### 🔧 如何设置数据同步

#### 第一步：获取JSONBin.io API密钥

1. 访问 [JSONBin.io](https://jsonbin.io)
2. 注册免费账户（无需信用卡）
3. 登录后，点击左侧菜单的 "API Keys"
4. 点击 "New API Key" 创建新密钥
5. 复制生成的API密钥（以"2_"开头的一长串字符）

#### 第二步：在应用中设置API密钥

1. 打开你的生活管理系统网页
2. 如果是首次访问，会自动弹出"数据同步设置"窗口
3. 将API密钥粘贴到输入框中
4. 点击"保存密钥"按钮
5. 等待验证完成

#### 第三步：验证同步

1. 在另一台设备或浏览器中访问你的应用
2. 使用相同的API密钥设置同步
3. 数据将自动同步到两台设备

### 🔐 数据安全

- **加密传输**：所有数据通过HTTPS加密传输
- **私密性**：只有拥有API密钥的人可以访问数据
- **本地备份**：数据同时保存在本地和云端
- **随时可控**：可以在JSONBin.io随时删除云端数据

### 🔄 同步机制

- **自动同步**：连接网络后每分钟自动检查并同步数据
- **手动同步**：点击右上角的同步状态按钮可立即同步
- **智能合并**：自动合并来自不同设备的数据，避免冲突
- **离线支持**：无网络时可正常使用，联网后自动同步

### 📱 多设备使用

1. **设置主设备**：首先在一台设备上设置API密钥并输入数据
2. **设置其他设备**：在其他设备上使用相同API密钥
3. **自动同步**：数据将在所有设备间保持同步

### 🛠️ 故障排除

#### 问题：同步失败
- 检查网络连接
- 验证API密钥是否正确
- 尝试手动同步

#### 问题：数据不匹配
- 检查所有设备是否使用相同的API密钥
- 尝试手动同步
- 确认数据已保存（刷新页面）

#### 问题：API密钥丢失
- 重新访问 [JSONBin.io](https://jsonbin.io) 获取新密钥
- 在所有设备上更新API密钥

### 💾 数据结构

同步的数据包括：
- 所有周计划
- 重要任务
- 快速任务
- 时间记录
- 应用设置

### 🌟 高级选项

#### 离线模式
如果你不想使用云端同步，可以在设置窗口选择"跳过（离线模式）"，数据将只保存在本地浏览器中。

#### 更换API密钥
1. 在应用中删除当前API密钥
2. 重新设置新的API密钥
3. 数据将开始同步到新的云端位置
