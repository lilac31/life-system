# 🚀 多端同步快速指南

## 📦 现在有两种同步方式

### 方式1：导入/导出（推荐，无限制）⭐⭐⭐⭐⭐

**优点**：
- ✅ 完全免费，无 API 限制
- ✅ 数据完全私有
- ✅ 支持任意设备
- ✅ 可通过任何方式传输（微信、邮件、U盘等）

**缺点**：
- ⚠️ 需要手动操作（不是自动实时同步）

**使用方法**：

#### 步骤1：导出数据（设备A）

1. 点击右上角的 **"📦 数据"** 按钮
   - 或按快捷键 `Ctrl/Cmd + Shift + E`

2. 在弹窗中点击：
   - **"下载文件"** - 下载 JSON 文件到本地
   - 或 **"显示数据"** 然后 **"复制"** - 复制到剪贴板

3. 将文件/数据发送到其他设备：
   - 微信/QQ 发送文件
   - 邮件发送
   - 网盘（百度云、坚果云等）
   - U盘拷贝
   - 或直接复制粘贴

#### 步骤2：导入数据（设备B）

1. 在设备B打开应用

2. 点击 **"📦 数据"** 按钮

3. 选择导入方式：
   - **"选择文件"** - 选择之前下载的 JSON 文件
   - 或直接 **粘贴** JSON 数据到文本框

4. 点击 **"导入数据"**

5. 等待页面刷新，数据同步完成！

#### 多端同步流程图

```
设备A                    传输方式                设备B
 │                                               │
 ├─ 导出数据 ──────────────────────────────────> │
 │   (下载/复制)      微信/邮件/网盘/U盘          │
 │                                               ├─ 导入数据
 │                                               │   (选择文件/粘贴)
 │                                               │
 │                                               ✅ 同步完成
```

## 💡 实用技巧

### 技巧1：定期备份

无论用哪种方式，都建议定期导出备份：

1. 每周导出一次
2. 保存到安全的地方（网盘、U盘）
3. 文件名：`backup-YYYY-MM-DD.json`

### 技巧2：快速传输

**电脑 → 手机**：
1. 导出数据 → 复制
2. 发送到微信文件传输助手
3. 手机打开 → 粘贴 → 导入

**电脑 → 电脑（同网络）**：
1. 使用 AirDrop（Mac）
2. 或局域网共享文件夹
3. 或微信/QQ 文件传输

### 技巧3：自动化（高级）

如果你会写脚本，可以：
1. 设置定时任务，每天自动导出
2. 上传到个人网盘
3. 其他设备定时下载导入
