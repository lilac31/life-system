# 个人日程管理系统 - 腾讯云快速部署指南

## 🚀 快速开始

### 前置条件
- 腾讯云账号（已实名认证）
- 已注册域名（可选）

### 部署步骤

#### 1. 构建前端应用

```bash
# 使用腾讯云专用配置构建
npm run build
```

#### 2. 创建腾讯云COS存储桶

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 进入 **对象存储** 服务
3. 创建新的存储桶：
   - **名称**：填写唯一的存储桶名称（如 `schedule-manager-2025`）
   - **地域**：选择就近的地域（如北京、上海、广州）
   - **访问权限**：选择 **公有读私有写**
   - **版本控制**：关闭
   - **服务端加密**：关闭

#### 3. 配置静态网站托管

1. 进入存储桶详情
2. 点击 **静态网站** 选项卡
3. 开启静态网站功能：
   - **索引文档**：`index.html`
   - **错误文档**：`index.html`
4. 保存设置

#### 4. 上传构建文件

将构建后的 `dist` 文件夹中的所有文件上传到存储桶根目录：

**方法一：网页上传**
- 点击 **上传文件** 按钮
- 依次上传所有文件（包括 `index.html` 和 `assets` 文件夹）

**方法二：使用 COSCLI（推荐）**
1. 安装 [COSCLI 工具](https://cloud.tencent.com/document/product/436/65951)
2. 配置密钥：
   ```bash
   coscli configure
   ```
3. 运行部署脚本：
   ```bash
   chmod +x deploy-tencent-cos.sh
   ./deploy-tencent-cos.sh
   ```

#### 5. 配置CDN加速

1. 进入 [CDN控制台](https://console.cloud.tencent.com/cdn)
2. 创建新的CDN域名：
   - **域名**：填写你的域名（如 `www.example.com`）
   - **源站类型**：COS
   - **源站地址**：选择刚创建的存储桶
   - **回源协议**：HTTPS
3. 点击 **创建**
4. 等待域名配置生效（约5-10分钟）

#### 6. 配置自定义域名

1. 在CDN域名列表中，点击刚创建的域名
2. 开启 **HTTPS**：
   - 选择 **免费证书**
   - 填写域名信息
   - 等待证书签发
3. 配置 **缓存规则**：
   - 添加规则：`index.html` 缓存时间 0 秒
   - 其他文件默认缓存 7 天

#### 7. 配置DNS解析

1. 进入 [域名解析控制台](https://console.cloud.tencent.com/cns)
2. 添加解析记录：
   - **记录类型**：CNAME
   - **主机记录**：@（或 www）
   - **记录值**：CDN提供的域名
   - **线路类型**：默认
   - **TTL**：600

## 🌐 访问你的应用

部署完成后，通过以下方式访问：

- **直接访问**：`https://你的CDN域名`
- **自定义域名**：`https://你的域名`

## 🔄 更新部署

当应用需要更新时：

```bash
# 1. 重新构建
npm run build

# 2. 上传新文件
./deploy-tencent-cos.sh

# 3. 在CDN控制台刷新缓存
# 选择你的域名 → 缓存刷新 → URL刷新 → 输入 https://你的域名/
```

## 🔧 故障排查

### 无法访问网站
1. 检查存储桶是否开启静态网站功能
2. 确认 `index.html` 在存储桶根目录
3. 检查CDN域名是否配置正确

### HTTPS证书不生效
1. 确认域名解析已生效
2. 检查CDN域名配置是否正确
3. 尝试刷新浏览器缓存

### 更新后页面未更新
1. 在CDN控制台执行缓存刷新
2. 尝试强制刷新浏览器（Ctrl+F5）

## 🎯 高级配置

### 自定义404页面
在存储桶根目录创建 `404.html` 文件，在静态网站设置中指定。

### 数据持久化
应用数据当前存储在浏览器本地存储中，如需云同步功能，可考虑：
1. 集成腾讯云数据库（MongoDB）
2. 使用腾讯云函数（SCF）提供API服务

### 访问统计
在CDN控制台开启访问分析，可以查看网站访问统计。
