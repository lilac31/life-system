# 数据同步快速开始指南

## 🎉 部署成功！

你的生活管理系统现已支持数据同步功能，可以在不同设备间保持数据一致。

## 🚀 快速开始

### 第一步：访问应用
打开网址：**https://lilac31.github.io/life-system**

### 第二步：设置数据同步
1. 首次访问会弹出"数据同步设置"窗口
2. 按照指南获取JSONBin.io API密钥
3. 输入密钥并点击"保存"

### 第三步：多设备同步
在其他设备上访问同一网址，使用相同的API密钥即可同步数据

## 📋 获取API密钥步骤

1. 访问 [JSONBin.io](https://jsonbin.io)
2. 注册免费账户（无需信用卡）
3. 登录后点击左侧菜单的"API Keys"
4. 点击"New API Key"创建密钥
5. 复制密钥（以"2_"开头的字符串）
6. 在应用中输入密钥

## ✅ 同步功能

- **自动同步**：连接网络后每分钟自动同步
- **手动同步**：点击右上角同步按钮
- **离线支持**：无网络时可正常使用
- **数据合并**：智能合并不同设备的数据

## 🔐 数据安全

- 使用HTTPS加密传输
- 只有持有API密钥者可访问
- 数据同时保存在本地和云端

## 💡 使用技巧

1. **设置提醒**：点击右上角同步状态查看最后同步时间
2. **手动同步**：如需立即同步，点击"立即同步"按钮
3. **数据备份**：数据自动保存在本地浏览器和云端

## 🌟 完成！

现在你可以在任何设备上访问应用，数据将自动同步，无需担心数据丢失！
