# 个人日程管理应用

一个现代化的个人日程管理应用，使用 React + Vite + Tailwind CSS 构建，具有美观的界面和完整的功能。

## 功能特性

### 📅 日历视图
- 月历显示，支持前一月/后一月切换
- 直观的任务展示，不同优先级用不同颜色标识
- 点击日期查看当天详细任务
- 任务完成状态一目了然

### 📝 任务管理
- 添加、编辑、删除任务
- 任务优先级设置（高/中/低）
- 任务分类管理（工作/个人/健康/学习）
- 任务完成状态切换
- 任务搜索和多维度筛选

### 🎨 用户界面
- 现代化的设计风格
- 响应式布局，支持各种屏幕尺寸
- 流畅的动画效果
- 直观的操作体验

### 🔧 技术特性
- React 18 + Hooks 开发
- Vite 构建工具，开发体验优秀
- Tailwind CSS 样式框架
- date-fns 日期处理库
- Lucide React 图标库

## 使用说明

### 添加任务
1. 点击侧边栏"添加任务"按钮或日历视图中的"+"按钮
2. 填写任务信息：标题、描述、日期、时间、优先级、分类
3. 点击"添加任务"完成创建

### 查看任务
- **日历视图**：在月历中查看任务分布，点击日期查看详情
- **任务列表**：查看所有任务的详细信息，支持搜索和筛选

### 管理任务
- 点击任务前的圆圈可标记完成/未完成
- 使用编辑按钮修改任务信息
- 使用删除按钮移除不需要的任务

### 筛选功能
在任务列表页面可以根据以下条件筛选：
- 搜索关键词
- 完成状态（全部/待完成/已完成）
- 优先级（全部/高/中/低）
- 分类（全部/工作/个人/健康/学习）

## 自定义配置

### 修改主题颜色
在 `tailwind.config.js` 中修改 `primary` 颜色配置

### 添加新的任务分类
1. 在 `Sidebar.jsx` 中的 `categories` 数组添加新分类
2. 在相关组件中更新 `getCategoryColor` 和 `getCategoryText` 函数

### 修改优先级选项
在 `AddTaskModal.jsx` 和相关组件中修改优先级选项和对应的样式函数

## 技术栈
